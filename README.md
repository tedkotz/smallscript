smallscript
===========

Small embeddable scripting langauge

Design Goals
------------
- JSON compatible object notation
- Small
    - Small executable footprint
    - Small memory footprint
    - Small-C compatible
    - Small language set
- Consistency
    - every thing should behave symmetrically
    - ranges should act like lists
- Inspirations
    - Duktape
    - Javascript
    - Lua
    - Python
